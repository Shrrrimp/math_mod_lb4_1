• Для перемещения объекта используйте стрелки на клавиатуре
  • Для поворота зажмите ctrl и стрелку
  • Для увеличения масштаба объекта зажмите shift и стрелку "вниз"
  • Для уменьшения масштаба объекта зажмите shift и стрелку "вверх"
  • Чтобы вернуть объект к исходному состоянию нажмите кнопку "Сброс"
